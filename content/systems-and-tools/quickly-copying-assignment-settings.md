# Quickly Copying Assignment Settings

For recurring classrooms, it can be quite laborious to set up the rubric, marking instructions, and submission content strategy for every assignment every week.

To make this easier, once you've synced all your assignments for a classroom, you can click the **"Copy settings"** button. Select the classroom you want to copy settings from and click **"Copy"**.

## Here's How It Works

For every coderbyte, Google, and GitHub assignment, the system looks for an assignment with a matching title. If it finds one, it copies the marking instructions, rubric, and submission content strategy over to your target classroom.

> Make sure to double check that everything copied over correctly before you run marking!
