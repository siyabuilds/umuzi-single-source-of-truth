# Creating Classes

## Steps to Create a Class

1. Go to the main Google Classroom page: [https://classroom.google.com/](https://classroom.google.com/)
2. Make sure you’re signed in with your **umuzi.org** account.
3. Click the plus icon at the top right and select the “Create class” option.

## Adding Class Details

- Add details about the class; only the class name is required. 
- The rest of the details, including the class name, can be changed later if needed.
- Click the “Create” button.

## Additional Information

- Add a short description under "Class description," ideally the cohort name.
- Add the cohort slug in the designated section.
- Be sure to invite admin users so the classroom doesn't get lost.
- All Skills Lab Classrooms should invite **technical@umuzi.org** as a teacher.
- XPL always has **experience.labs@umuzi.org** as a teacher, which means that multiple people can access any classroom if needed.
