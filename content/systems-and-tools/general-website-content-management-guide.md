# General Website Content Management Guide

## User Role Overview

- **Editor (Non-Technical Staff)**: Can change text, images, and blog posts. They cannot change the layout, colors, or fonts.
- **Admin (Web Lead/Developer)**: Can change everything, including the structure, styling, and site settings.

## 1. Managing CMS Content (Blog, Team members, etc.)

The CMS (Content Management System) is like a database/spreadsheet that powers the site. The CMS is mainly for *repetitive* content.

### For Editors (Standard Updates)

#### To Add: 
1. Open the Collections tab (stack of coins icon)
2. Pick a category 
3. Click + New Item 
4. Fill in the fields 
5. Click Create.

#### To Update: 
1. Click an existing item in the list 
2. Edit the fields
3. Click Save.

#### To Remove: 
1. Open the item 
2. Scroll to the bottom 
3. Click Delete (or Archive to hide it without losing the data).

### For Admins (Structural Changes)

- **Add New Fields**: If the team needs a new "Date" or "Author" field in the CMS, only an Admin should add this in the Designer to ensure it connects to the page layout correctly.

## 2. Managing Static Content (Headlines, About Us, Buttons)

This is for content that isn't part of a CMS collection list or database.

### For Editors (Visual Editing)

#### Update Text/Images: 
1. While on the live-preview page, hover over the text or image and click the Edit icon that appears 
2. Type your changes or upload a new photo.

#### Update Buttons: 
1. Click the button 
2. Click the Settings (Gear) icon
3. Change the URL link.

> **What you CAN'T do**: Editors cannot move a button to a different part of the page or change its color.

### For Admins (Layout Changes)

- **Add/Remove Sections**: If a whole new paragraph or image block is needed, an Admin must drag in a new Element from the Add panel ($+$).
- **Deleting Content**: Admins should handle deleting sections to ensure the "empty space" (padding/margin) is adjusted so the site still looks good on mobile.

## 3. Publishing Changes

- **Editors**: When you finish an update, click the Publish button in the bottom bar. This only updates the specific content you changed.
- **Admins**: When making layout changes in the Designer, you must use the top-right Publish menu. Be sure to check both the "Staging" and "Production" boxes.
