# Adding and Publishing Different Types of Class Content

## Organization of Class Content

“Topics” are the built-in way to organize class content in Google Classroom. That doesn’t mean you can workaround it and customize the organization of content though. Hopefully, all classes will come with a building/organization brief like this one.

> To add a topic, see **Adding content to a class** below. 

Note that topics need to have published content to be visible to students; otherwise, they’ll only be visible to the teachers of a class.

## Adding Content to a Class

1. Go to the **classwork** tab.
2. Click the **Create** button.
3. Choose the type of content you want to add.

## Type of Content You Can Add

- **Topics** - sections of a class. They’re not actual content but used for organizing the actual content.
- **Material** - content that doesn’t require any submission from the student.
- **Assignments** - content that requires the learner to submit something.
