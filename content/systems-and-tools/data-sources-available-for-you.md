# Data Sources Available for You

## Dashboards
We have two org-wide high level data dashboards available:
- **Recruitment Dashboard**: Showing data about the recruitment process & applications.
- **Enrolment Dashboard**: Showing data about current learners across the Labs.

These dashboards speak to recruitment data since our **3rd March 2025** relaunch and enrolment data for **2025** respectively.

## Data Sources
To run your operational processes, you need access to certain data about learners. As guidelines to meet our Data Regulations, please make sure that you are working with data in the relevant platforms primarily to see and check what you need to. 

> You should only export data if required to do what you need. If you do export data into a sheet, please ensure that this is shared with only the people who need access and that the sheet is archived or deleted once its purpose is served. If we need to retain that data for longer, it should be shared with the Data Team in order to save that data in a centralized place.

> Learner data should never be shared with external parties unless covered in the data agreement learners have agreed to and covered by a Data Processing Agreement with the external party.
