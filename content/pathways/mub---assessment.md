# MUB - Assessment

## AIM
The intention of this document is to assist and guide a facilitator in the assessment of the online Design Thinking sprint which occurs at the beginning of the learner's journey called - **MUB (Make Us Better)**.

There are two components to this assessment SOP:
- Video demonstration
- Step-by-step guide

## MUB ASSESSMENT - VIDEO DEMONSTRATION
This is a video demonstration of how to view the learners' submissions, access the rubric as a Google Doc, and how best to assess each work. It can be watched alongside the accompanying step-by-steps.

## STEP-BY-STEP GUIDE
This is a step-by-step guide for how to view the learners' submissions, access the rubric as a Google Doc, and how best to assess each submission.

### STEP 1: SUBMISSION LINKS
First, you will need to check that each group has submitted their required links on the MUB Airtable Base:
1. Deliverable & team tab
2. Click on grid view (left)
3. Search a name for one member 
4. Click the link (will go to the file)

Links to assess:
- Link to Figma File
- Link to Presentation (If different from the above Figma file)
- Link to Video of presentation 

Check that the permissions for each file allow for you, as the assessor, to have access.

### STEP 2: ASSESSMENT
You will be required to duplicate the template, per learner, and assess each requirement per submission which is represented on each tab.

Here is the link to the assessment - template. 

Make a duplicate or copy of this template for each of the groups you will be assessing. Name each duplicate assessment sheet using the following format: `Cohort#_Group#_MUB - Assessment`.

1. Open a file to assess the submission.
2. Go to the relevant tab for that submission (see image below) - e.g., Final presentation.
3. Grade each submission using the prescribed requirements based on the quality of work submitted. The grades can range from 0 - 100%.

The scale of assessment for grading is as follows:
- Add additional comments per submission at the bottom of the assessment requirements. 
- Once completing the assessment of all submissions and requirements, the comments and scores will auto-populate to the Summary tab.

Repeat for all files and submissions that need to be assessed per group and learner.
