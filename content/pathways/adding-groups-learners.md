# Adding Groups & Learners

This is how to create new groups and add new learners to groups. Each learner should be linked to at least one group and collection for reporting purposes:

- Each learner must be linked to one (and only one) group on the table **[Groups]**; this determines the learner's start and end dates for reporting (e.g. YomAKenya0106).
- Learners can be linked to more than one group on **[Groups +]**, which allows us to include learners in multiple reports (e.g. Data Analytics), without interfering with their start and end dates for reporting.
- The view **[Learners] > Data Cleaning > No Group** should display learners without a group.
- New learners will appear in the data imported from Coursera; they need to be allocated to a group.
- Learners also need to be linked to a Collection. Check the view **[Learners] > Data Cleaning > Collection is empty**.
- It can be useful to look at what courses a learner has started/completed to check they are allocated to the correct collection.

## To Create a New Group:

1. On **[Groups]**, add a record with the new Group name, where possible, following the standard group name formats:
   - New external ID: e.g. YoMaNigeria0106
2. On **[Groups +]**, add a record with the new Group name, where possible, following the standard group name formats:
   - Group starting on a collection on a certain date: YYMMDD - Collection name
3. Add start date and length of programme (in weeks).
